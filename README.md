# Conversor de Moedas em Java

Projeto desenvolvido como desafio do programa Oracle Next Education + Alura.

## Funcionalidades

- Conversão de moedas em tempo real
- Consumo de API de câmbio
- Interface via console
- Menu interativo

## Tecnologias utilizadas

- Java 17
- Gson
- HttpClient
- ExchangeRate API

## Moedas suportadas

- BRL (Real Brasileiro)
- USD (Dólar Americano)
- ARS (Peso Argentino)
- CLP (Peso Chileno)

## Como executar

1. Clone o repositório
2. Insira sua chave da API no arquivo `ApiExchange.java`
3. Execute o arquivo `Main.java`
